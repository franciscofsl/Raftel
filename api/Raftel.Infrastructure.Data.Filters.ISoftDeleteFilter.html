    Interface ISoftDeleteFilter

    Marker interface used to identify the soft delete filter in the data filtering system.

    Namespace: Raftel.Infrastructure.Data.Filters
    Assembly: Raftel.Infrastructure.dll
    Syntax
    public interface ISoftDeleteFilter
    Remarks

    This interface has no members and is used exclusively as a type token to represent the soft delete filter when enabling or disabling it at runtime via IDataFilter.
